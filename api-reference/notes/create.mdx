---
title: Create note
description: Create a new note for the authenticated user.
---

Creates a new note and returns the created object. Only `title` is required; all other fields are optional.

## Request body

<ParamField body="title" type="string" required>
  Note title. Must be a non-empty string.
</ParamField>

<ParamField body="content" type="string">
  Rich text HTML content for the note body.
</ParamField>

<ParamField body="isPinned" type="boolean" default="false">
  Whether the note should be pinned.
</ParamField>

<ParamField body="isArchived" type="boolean" default="false">
  Whether the note should be archived immediately on creation.
</ParamField>

<ParamField body="background" type="string">
  Background color or pattern identifier to apply to the note.
</ParamField>

<ParamField body="tagIds" type="string[]">
  Array of tag IDs to assign to the note. Each ID must correspond to a tag owned by the authenticated user.
</ParamField>

## Response

Returns the created Note object.

<ResponseField name="id" type="string" required>
  Unique identifier (UUID) of the created note.
</ResponseField>

<ResponseField name="title" type="string" required>
  Note title.
</ResponseField>

<ResponseField name="content" type="string">
  Rich text HTML content. `null` if not provided.
</ResponseField>

<ResponseField name="isPinned" type="boolean" required>
  Whether the note is pinned.
</ResponseField>

<ResponseField name="isArchived" type="boolean" required>
  Whether the note is archived.
</ResponseField>

<ResponseField name="background" type="string">
  Background identifier. `null` if not provided.
</ResponseField>

<ResponseField name="state" type="string" required>
  Always `active` for newly created notes.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of creation.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last update.
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the user who owns this note.
</ResponseField>

<ResponseField name="tags" type="object[]" required>
  Array of tag objects assigned to this note.
</ResponseField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://anchor.example.com/api/notes' \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "title": "Project ideas",
    "content": "<p>Some initial thoughts...</p>",
    "isPinned": true,
    "tagIds": ["tag-uuid-1", "tag-uuid-2"]
  }'
```

</RequestExample>
