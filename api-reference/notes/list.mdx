---
title: List notes
description: Retrieve all active notes for the authenticated user, with optional filtering by search query, tag, or result limit.
---

Returns an array of active notes (state: `active`) belonging to the authenticated user. Trashed and archived notes are excluded from this endpoint — use [List trashed notes](/api-reference/notes/trash) or [List archived notes](/api-reference/notes/archive) for those.

## Query parameters

<ParamField query="search" type="string">
  Filter notes by title or content. Case-insensitive substring match.
</ParamField>

<ParamField query="tagId" type="string">
  Filter notes by tag ID. Returns only notes that have the specified tag assigned.
</ParamField>

<ParamField query="limit" type="number">
  Maximum number of notes to return. When omitted, all matching notes are returned.
</ParamField>

## Response

Returns an array of Note objects.

<ResponseField name="[]" type="object[]">
  <Expandable title="Note properties" defaultOpen>
    <ResponseField name="id" type="string" required>
      Unique identifier (UUID).
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Note title.
    </ResponseField>
    <ResponseField name="content" type="string">
      Rich text HTML content. `null` if no content has been set.
    </ResponseField>
    <ResponseField name="isPinned" type="boolean" required>
      Whether the note is pinned.
    </ResponseField>
    <ResponseField name="isArchived" type="boolean" required>
      Whether the note is archived.
    </ResponseField>
    <ResponseField name="background" type="string">
      Background color or pattern identifier. `null` if not set.
    </ResponseField>
    <ResponseField name="state" type="string" required>
      Note state. One of `active`, `trashed`, or `deleted`.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp of when the note was created.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
    <ResponseField name="userId" type="string" required>
      ID of the user who owns this note.
    </ResponseField>
    <ResponseField name="tags" type="object[]" required>
      Array of tags assigned to this note.
    </ResponseField>
  </Expandable>
</ResponseField>

```bash cURL
curl --request GET \
  --url 'https://anchor.example.com/api/notes?search=meeting' \
  --header 'Authorization: Bearer <token>'
```
