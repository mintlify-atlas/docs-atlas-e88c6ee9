---
title: List archived notes
description: Retrieve all archived notes for the authenticated user.
---

Returns all notes where `isArchived` is `true` for the authenticated user. Archived notes remain accessible and searchable but are separated from the main notes list.

<Note>
  Archived notes are excluded from [List notes](/api-reference/notes/list). Use this endpoint to power an archive view. To unarchive a note, use [Update note](/api-reference/notes/update) and set `isArchived: false`.
</Note>

## Response

Returns an array of Note objects with `isArchived: true`.

<ResponseField name="[]" type="object[]">
  <Expandable title="Note properties" defaultOpen>
    <ResponseField name="id" type="string" required>
      Unique identifier (UUID).
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Note title.
    </ResponseField>
    <ResponseField name="content" type="string">
      Rich text HTML content. `null` if not set.
    </ResponseField>
    <ResponseField name="isPinned" type="boolean" required>
      Whether the note is pinned.
    </ResponseField>
    <ResponseField name="isArchived" type="boolean" required>
      Always `true` for notes returned by this endpoint.
    </ResponseField>
    <ResponseField name="background" type="string">
      Background color or pattern identifier. `null` if not set.
    </ResponseField>
    <ResponseField name="state" type="string" required>
      Note state: `active`, `trashed`, or `deleted`.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 creation timestamp.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
    <ResponseField name="userId" type="string" required>
      ID of the owning user.
    </ResponseField>
    <ResponseField name="tags" type="object[]" required>
      Tags assigned to this note.
    </ResponseField>
  </Expandable>
</ResponseField>

```bash cURL

  --header 'Authorization: Bearer <token>'
```
