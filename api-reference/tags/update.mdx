---
title: Update tag
description: Update an existing tag, retrieve a single tag, or list all notes associated with a tag.
---

## Update tag

`PATCH /api/tags/:id`

Partially updates an existing tag. Only the fields included in the request body are changed.

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the tag to update.
</ParamField>

### Request body

All body fields are optional.

<ParamField body="name" type="string">
  New tag name. Must be unique for the authenticated user.
</ParamField>

<ParamField body="color" type="string">
  New color identifier or hex value.
</ParamField>

### Response

Returns the updated Tag object.

<ResponseField name="id" type="string" required>
  Unique identifier (UUID).
</ResponseField>

<ResponseField name="name" type="string" required>
  Tag name.
</ResponseField>

<ResponseField name="color" type="string">
  Color identifier. `null` if not set.
</ResponseField>

<ResponseField name="isDeleted" type="boolean" required>
  Deletion status of the tag.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 creation timestamp.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of this update.
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the owning user.
</ResponseField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url 'https://anchor.example.com/api/tags/tag-uuid-1' \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "color": "#EF4444"
  }'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "tag-uuid-1",
  "name": "work",
  "color": "#EF4444",
  "isDeleted": false,
  "createdAt": "2024-01-10T09:00:00.000Z",
  "updatedAt": "2024-03-15T11:00:00.000Z",
  "userId": "user-uuid-here"
}
```

</ResponseExample>

## Get tag

`GET /api/tags/:id`

Retrieves a single tag by its ID.

<ParamField path="id" type="string" required>
  The UUID of the tag to retrieve.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://anchor.example.com/api/tags/tag-uuid-1' \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "tag-uuid-1",
  "name": "work",
  "color": "#3B82F6",
  "isDeleted": false,
  "createdAt": "2024-01-10T09:00:00.000Z",
  "updatedAt": "2024-01-10T09:00:00.000Z",
  "userId": "user-uuid-here"
}
```

</ResponseExample>

## Get notes by tag

`GET /api/tags/:id/notes`

Returns all notes that have the specified tag assigned.

<ParamField path="id" type="string" required>
  The UUID of the tag.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://anchor.example.com/api/tags/tag-uuid-1/notes' \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>

<ResponseExample>

```json 200
[
  {
    "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "title": "Work planning",
    "content": "<p>Q2 goals...</p>",
    "isPinned": true,
    "isArchived": false,
    "background": null,
    "state": "active",
    "createdAt": "2024-02-01T09:00:00.000Z",
    "updatedAt": "2024-03-01T10:00:00.000Z",
    "userId": "user-uuid-here",
    "tags": [
      { "id": "tag-uuid-1", "name": "work", "color": "#3B82F6" }
    ]
  }
]
```

</ResponseExample>
