---
title: List tags
description: Retrieve all active tags for the authenticated user.
---

Returns all tags owned by the authenticated user that have not been soft-deleted (`isDeleted: false`).

## Response

Returns an array of Tag objects.

<ResponseField name="[]" type="object[]">
  <Expandable title="Tag properties" defaultOpen>
    <ResponseField name="id" type="string" required>
      Unique identifier (UUID).
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Tag name. Unique per user.
    </ResponseField>
    <ResponseField name="color" type="string">
      Color identifier or hex value. `null` if not set.
    </ResponseField>
    <ResponseField name="isDeleted" type="boolean" required>
      Always `false` for tags returned by this endpoint.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 creation timestamp.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of the last update.
    </ResponseField>
    <ResponseField name="userId" type="string" required>
      ID of the user who owns this tag.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://anchor.example.com/api/tags' \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>
